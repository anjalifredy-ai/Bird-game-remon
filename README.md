# Bird Game 🐦

Flutter + Flame based casual mobile game. Built entirely via GitHub mobile app + GitHub Actions (no local machine needed).

## Stage 1 — Skeleton
Proves the build pipeline works. Push all files → check Actions tab → download APK from Artifacts.

## Roadmap
- Stage 2: Core game engine (bird physics, pipes, coins, difficulty)
- Stage 3: Main Menu screen
- Stage 4: Mode Selection screen
- Stage 5: Game Over screen
- Stage 6: Settings + sound manager
- Stage 7: Assets guide
- Stage 8: Final wiring + polish
